--- 2. Budget auto-apply logic in App.jsx ---

Add inside the existing recurring useEffect (after app becomes ready),
after the recurring generation block:

  // Auto-apply budget template if needed
  const currentMonth = today().slice(0, 7);
  const hasCurrentBudgets = budgets.some(b => b.month === currentMonth);
  if (!hasCurrentBudgets) {
    const autoTemplate = budgetTemplates.find(t => t.autoApply);
    if (autoTemplate) {
      const newBuds = autoTemplate.budgets.map(b => ({ ...b, id: uid(), month: currentMonth }));
      setBudgets(prev => [...prev, ...newBuds]);
    }
  }


--- 3. onToggleTemplateAutoApply handler ---

Add to App.jsx:
  const handleToggleTemplateAutoApply = (name) =>
    setBudgetTemplates(prev => prev.map(t => t.name === name ? { ...t, autoApply: !t.autoApply } : t));

Pass to Budgets render:
  <Budgets
    ...existingProps...
    onToggleTemplateAutoApply={handleToggleTemplateAutoApply}
  />
